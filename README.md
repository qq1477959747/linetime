# LineTime - 时间线应用

一个支持个人与多人共享的时间线应用，用户可以在特定日期上传照片、编写文字记录生活点滴。

## 项目特点

- 📅 个人时间线 + 共享空间
- 📸 支持多图上传与管理
- 💑 情侣空间功能
- 🔗 邀请码/链接分享机制
- 🎨 现代化的 UI 设计

## 技术栈

### 前端
- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS + shadcn/ui

### 后端
- Go 1.21+
- Gin Web Framework
- GORM
- PostgreSQL 15+
- Redis 7+
- MinIO
